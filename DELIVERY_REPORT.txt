## 🚀 How to Get Started

### Step 1: Read Documentation (5 minutes)
```
Start with: START_HERE.md (visual overview)
Then: SETUP.md (installation guide)
```

### Step 2: Install & Configure (10 minutes)
```bash
composer install
npm install
cp .env.example .env
# Add your Turso & Cloudinary credentials
php artisan key:generate
php artisan migrate
```

### Step 3: Launch (5 minutes)
```bash
npm run dev    # Terminal 1
php artisan serve    # Terminal 2
# Visit http://localhost:8000
```
